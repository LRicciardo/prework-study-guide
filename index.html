Prework Study Guide

HTML

  • Acronym: HyperText Markup Language
  • File extension(s): .html or .htm
  • HTML gives the content structure.
  • There are 4 types of structural and data elements: Text, Image, Header and List
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are 3 types of content elements: Meta-related, Content division, and Semantic.
  • The meta-related elements defines metadata such as the character set, description, keywords, author, and viewport.

CSS

  • Acronym: Cascading Style Sheets
  • File extension(s): .css
  • CSS gives the content style.

Git

  • "Git is a version control system that lets you manage and keep track of each version of your project."
  • GitHub is a cloud-based hosting service that allows us to keep track of our projects inside repositories.
  • command: git status => shows files in working directory
  • command: git checkout -b branch-name => creates a new branch and switches to new branch
  • command: git checkout branch-name => switches to that branch
  • command: git pull origin branch-name => recieve and merge any commits from the tracking remote branch
  • command: git push origin branch-name => sends local branch commits to remote repository branch
  • command: git commit -m "text" => creates versions that need to be merged with message ??
  • command: git add -A => adds or includes files to branch currently attached
  • side note: command: code . => starts the Visual Studio (VS) Code editor

JavaScript

  • Abbreviation: JS
  • File extension(s): .js, .cjs, .mjs
  • JS handles logic and allows the user to interact with a website.
